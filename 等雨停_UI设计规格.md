# 《等雨停》UI 设计规格

## 1. 设计目标

本 UI 设计稿用于表达《等雨停》场景化交互的界面层级。核心原则是：场景物件负责触发互动，UI 元件只负责状态反馈、轻提示和输入承接。

背景图可先使用 16:9 占位符，后续替换为正式雨夜房间图。

## 2. 画布规格

| 项目 | 规格 |
|---|---|
| 画布比例 | 16:9 |
| 推荐尺寸 | 1536 × 864 |
| 安全区 | 左右 40px，上下 32px |
| 背景 | 全屏铺满，使用 `cover` 裁切 |
| 遮罩 | 顶部和底部保留暗角，提高 UI 可读性 |

## 3. 总体布局

### 3.1 左上：天气进度模块

位置：

- x: 54
- y: 42
- 宽度: 约 500
- 高度: 约 156

内容：

- 标题：天气进度
- 四阶段图标：雷雨中 / 小雨 / 雨停 / 天晴
- 横向进度线
- 当前阶段高亮
- 当前天气描述文案

说明：

天气进度模块不可点击，只展示状态。当前阶段使用发光图标和高亮文字，后续阶段使用低透明度灰白色。

### 3.2 上方偏中：放晴值

位置：

- x: 550
- y: 56

内容：

- `放晴值`
- 增量数值，例如 `+24`

状态：

- 默认隐藏或弱显示。
- 用户完成有效互动后出现。
- 1.5-2 秒后淡出。

视觉：

- Label 使用白色。
- 增量数值使用绿色 `#78DFA0`。

### 3.3 左侧中部：天气变化 Toast

位置：

- x: 38
- y: 218
- 宽度: 322
- 高度: 44

内容：

- 小天气图标
- 文案：如“雨声好像轻了一点。”
- 消失提示：如“2s后消失”

视觉：

- 背景：`rgba(18, 27, 43, 0.58)`
- 圆角：22
- 边框：白色 12%-16% 透明度
- 文案颜色：`#F1F6FF`
- 次级信息：`#AEB7C8`

交互：

- 不可点击。
- 每次有效互动后展示。
- 阶段切换时优先展示阶段切换提示。

## 4. 场景热点

### 4.1 作用

提示当前场景中可交互的物件。热点不能像按钮，应像环境中的微弱光点。

### 4.2 推荐位置

以 1536 × 864 画布为基准：

| 热点 | x | y | 说明 |
|---|---:|---:|---|
| 风铃 | 850 | 156 | 点击后风铃摇晃，雨声变轻 |
| 角色 | 504 | 512 | 未坐下时回头，坐下后倒茶 |
| 旧相册 | 101 | 783 | 触发老照片记忆 |
| 茶杯 | 822 | 763 | 坐下后可喝茶 |
| 石板路 | 1125 | 784 | 180 秒后可走出去 |

### 4.3 视觉规格

| 层级 | 规格 |
|---|---|
| 外层光晕 | 直径约 74px，蓝色低透明度，模糊 8-14px |
| 光圈 | 直径 42px，描边 1.5px，颜色 `#BBD0FF` |
| 中心光点 | 直径 10px，颜色 `#EEF4FF`，带蓝色发光 |
| 随机粒子 | 直径 5px，不定时浮现 |

动效：

- 光晕 3-5 秒呼吸。
- 光圈透明度缓慢变化。
- 随机粒子间歇出现，避免持续闪烁。

状态：

- 可交互：正常光圈。
- 推荐交互：光圈更亮。
- 条件未满足：光圈更弱，点击后角色解释。
- 已完成：光圈淡出或只保留极弱光点。

## 5. 聊天输入模块

### 5.1 位置

- 水平居中
- 底部距离约 45px
- 宽度: 430
- 高度: 126

### 5.2 结构

输入模块包含：

- 外层玻璃面板
- 文本输入框
- 发送图标
- 解锁提示文案

### 5.3 未解锁状态

文案：

- 输入框占位：“如果想说，写一句就好...”
- 底部提示：“喝过茶后，想说的话就会被听见。”

视觉：

- 外层背景：`rgba(16, 24, 39, 0.66)`
- 输入框背景：`rgba(18, 27, 43, 0.64)`
- 边框：白色 16%-18% 透明度
- 圆角：外层 18，输入框 10

交互：

- 未解锁时点击输入框不弹键盘。
- 可以轻微提示用户先喝茶。

### 5.4 已解锁状态

触发条件：

1. 用户已坐下。
2. 用户已喝茶。

状态变化：

- 底部锁定提示消失。
- 输入框亮度略微提升。
- placeholder 改为：“不用完整，几个字也可以。”

## 8. 组件状态汇总

### 天气进度

| 状态 | 表现 |
|---|---|
| 当前阶段 | 图标发光、文字高亮、进度线点亮 |
| 未到阶段 | 图标灰白低透明度、文字弱化 |
| 阶段切换 | 图标短暂增强发光，进度线推进 |

### 放晴值

| 状态 | 表现 |
|---|---|
| 默认 | 隐藏 |
| 互动后 | 显示 `+N`，绿色，高亮 |
| 消失 | 1.5-2 秒淡出 |

### Toast

| 状态 | 表现 |
|---|---|
| 默认 | 隐藏 |
| 出现 | 互动后显示天气变化文案 |
| 阶段切换 | 优先展示“大雨转小了”等阶段文案 |
| 消失 | 2-3 秒自动淡出 |

### 热点

| 状态 | 表现 |
|---|---|
| 可交互 | 蓝色光圈和中心光点 |
| 推荐交互 | 光圈更亮，呼吸更明显 |
| 条件未满足 | 光圈更弱 |
| 已完成 | 光圈淡出或消失 |

### 输入框

| 状态 | 表现 |
|---|---|
| 未解锁 | 可见但弱化，底部展示锁定提示 |
| 可输入 | 输入框亮度提升，锁定提示消失 |
| 输入中 | 面板边框略亮 |
| 发送后 | 输入框收起或恢复弱态，角色气泡显示回复 |
